# Power BI Retail Revenue Dashboard

This folder contains the three page Power BI dashboard built on the synthetic Superstore dataset generated in the main project. The visuals provide KPI monitoring, revenue breakdowns, and trend analysis for a retail environment using a clean navy and gold executive theme.

---

## Dashboard Pages

### 1. KPI Summary
- Total Revenue  
- Total Orders  
- Average Order Value  
- Total Profit  
- Interactive date slicer  
Designed for quick executive level scanning and high impact insights.

### 2. Revenue Breakdown
- Revenue by Segment  
- Revenue by Category  
- Revenue by Region  
Provides a clear comparison of performance across business dimensions and customer groups.

### 3. Trend Analysis
- Daily Revenue Trend  
- Profit Trend  
- Optional Category slicer  
Helps identify seasonal patterns, behavior shifts, and operational trends over time.

## Public Access Notice

Power BI Service does not allow public sharing or generating a "publish to web" link for this report under the current account settings. As a result, the interactive online version cannot be shared publicly.

To ensure full transparency and reproducibility, this folder includes:

- The full `.pbix` report file (`retail_revenue_dashboard.pbix`)
- Screenshots of each dashboard page
- The dataset used to build the report (`../data/superstore_sample_large.csv`)
- Instructions on how to refresh and explore the dashboard locally using Power BI Desktop

Anyone with Power BI Desktop can open the `.pbix` file and fully interact with the report locally.
